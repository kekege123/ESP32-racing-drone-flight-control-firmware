# ESP32-racing-drone-flight-control-firmware
This is a standard PlatformIO project, under the Arduino framework, based on the standard ESP32 main control for quadcopter flight control firmware, implementing dual-loop control from angle loop to angular velocity loop.
Integrated voltage monitoring, buzzer music, GPS data and azimuth, display of analog video transmission OSD information, ELRS receiver signal analysis, and multiple functions such as TF card black box storage.
